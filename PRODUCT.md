# Product

## Register

product

## Users
Authors, editors, proofreaders, and readers. They use the platform to share manuscripts, perform editorial reviews with comments, and consume books in a secure, often offline, web-based environment.

## Product Purpose
`do EPUB Studio` is a production-grade web platform for creating, managing, and reading EPUB files. It aims to provide a secure editorial workspace that bridges the gap between private distribution and collaborative review.

## Brand Personality
- **Intentional**: Every interaction and transition is purposeful (2026 standards).
- **Secure**: Industrial-grade protection without compromising usability.
- **Reliable**: "Offline-first" mindset ensures the work continues anywhere.

## Anti-references
- **Generic SaaS Clichés**: Avoid "ghost cards" (1px border + massive shadow) and over-rounded corners (>16px).
- **AI Slop**: No generic "eyebrow" section labels or aphoristic-cadence copy.
- **Low Contrast**: No muted gray text on cream backgrounds; maintain high legibility for long-form reading.

## Design Principles
1. **Security as Utility**: Security features like signed URLs and sanitization should be invisible to the user but omnipresent.
2. **Reading First**: The core experience is long-form text consumption; UI should recede to support focus.
3. **Local-First, Sync-Second**: Ensure the interface is always responsive by prioritizing local state and IndexedDB persistence.

## Accessibility & Inclusion
- Target WCAG 2.1 Level AA compliance.
- Support for high-contrast themes and reduced motion.
- OKLCH color space for perceptual uniformity across different displays.
