Several issues overlap multiple perspectives, underscoring systemic weaknesses around access control, signed downloads, and password storage. - **Testing:** Meaningful worker and E2E tests (G10, G11) rely on schema validation (G12) and trusted auth flows (G4, G5) so they can assert deterministic outcomes.
- **Annotation reliability:** Enforcing multi-signal locators (part of G13) requires shared schema enforcement (G12) and book scoping fixes within the same endpoints.
