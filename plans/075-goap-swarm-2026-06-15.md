  write; admin can recover and edit books; reader resumes from
  last known progress; magic-link email actually delivers;
  schema validation is centralized; ADR hygiene is restored.
- **Constraints:** All work must pass `./scripts/quality_gate.sh`,
  ship behind atomic commits, follow AGENTS.md TIER-1 rules
  (Argon2id, signed URLs, multi-signal locators, semantic design
  tokens, ReDoS safety).
- **Complexity:** Complex. Spans DB schema, Worker API routes,
  React UI, i18n, tests (unit + E2E), docs.

- **G14/G22 are tenant-isolation changes** — the existing reader
  flow must continue to work; rollback is a single revert.

  to "no-op + audit" in dev until production secrets are set.

  store; can be disabled in `preferences.ts` without redeploy.
- **G20 Zod centralization** — backward-compatible: routes keep
  the same exported names, just import from the schema package.
